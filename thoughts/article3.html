# 从 yml 配置文件到 css juejin

# 背景描述

最近用vue制作项目，遇到了一个问题。我想描述自己掌握的技术，以及相应的「掌握程度」，但又不想挨个 css 修改。于是我想，能否实现自动读取配置文件中的数值，生成与「掌握程度」相对应的样式呢？

语言描述起来有些许抽象，我们直接上图吧。

左图是对各项技能的掌握程度，右图则是期望的效果。

demo地址：https://github.com/dandanDQ/wonderful-demos

# 方案介绍

使用 yaml 文件作为配置，输入技术名称及对应的掌握程度。

vue 文件中读取 yaml 中设置的数值，自动生成样式。

# 2. 读取 yaml 文件数据，并作为 vue 选项输出
// getVueOptions.js
import yaml from 'js-yaml';
import {
    PERSON
} from './data.yml';

function getVueOptions (name) {
    const opt = {
        name: name,
        data () {
            return {
                person: yaml.load(PERSON),
            };
        },
    };
    return opt;
}

export {
    getVueOptions
};
  1. vue 模板文件
<template>
  <div class="skills-block left-module">
    <h3 class="title">{{ person.lang === 'en'?  'Skills' : '个人技能'}}</h3>
    <div class="skills">
      <span class="skill" v-for="skill in person.skills" :style="{'--percentage': (100-skill.level)+'%'}" >
        <span class="skill-name">{{skill.name}}</span>
      </span>
    </div>
  </div>
</template>
<script>
import Vue from 'vue';
import { getVueOptions } from './options';

const name = 'yaml-2-css';
export default Vue.component(name, getVueOptions(name));
</script>

# 核心知识点

yaml文件读取 YAML 语言教程 安装js-yaml： npm install --save js-yaml

css变量使用与clip-path结合使用实现自动高度显示：

clip-path: polygon(0 var(--percentage), 100%  var(--percentage), 100% 100%, 0 100%);

安装less：npm install less less-loader --save-dev
