# 关于 vue 的响应式 reactivity

相关文章： 响应式原理 vue3 响应式原理

Vue will walk through all of its properties and convert them to Proxies (opens new window)using a handler with getters and setters.

dependency-tracking and change-notification

tracked a list of dependencies — the properties it accessed during the render the component becomes a subscriber to each of these properties When a proxy intercepts a set operation, the property will notify all of its subscribed components to re-render.

proxy消除的局限： 1 对属性的添加、删除动作的监测； 2 对数组基于下标的修改、对于 .length 修改的监测； 3 对 Map、Set、WeakMap 和 WeakSet 的支持；

weakmap 不会有垃圾清除的问题
